# To-Do List App

A step-by-step Python project that started as a simple CLI To-Do List and will grow into a more advanced task manager and time tracker application.

## Current Version & Roadmap

This version includes:
- [x] Basic CLI menu
- [x] Add task
- [x] View tasks
- [x] Complete task
- [x] Delete task
- [x] Save tasks to file
- [x] Load tasks from file
- [x] Input validation for empty tasks
- [x] Error handling for invalid task numbers

### Next Steps
- [ ] Edit existing tasks
- [ ] Add task priorities
- [ ] Add due dates
- [ ] Filter completed and uncompleted tasks
- [ ] Search tasks
- [ ] Sort tasks by priority or due date
- [ ] Clear all tasks
- [ ] Add categories for tasks
- [ ] Add timer / Pomodoro feature
- [ ] Track time spent on tasks
- [ ] Build GUI version
- [ ] Build web version
## Technologies Used
- Python
- File handling
- Command-line interface (CLI)
